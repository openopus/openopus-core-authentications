## Installation
Add this line to your application's Gemfile:

```ruby
gem 'openopus-core-people'
gem 'openopus-core-authentications'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install openopus-core-people
$ gem install openopus-core-authentications
```
